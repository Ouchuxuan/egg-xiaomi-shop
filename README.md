# egg仿小米商城
后台使用阿里egg.js，mongoose开发的仿小米商城全栈项目


### 主要依赖
+ egg.js
+ ejs.js
+ mongoose

#### 本地开发
```bash
$ npm i
$ npm run dev
$ open http://localhost:8090/
```

### 部署

```bash
$ npm start
$ npm stop
```

### 内置指令

- 使用 `npm run lint` 来做代码风格检查。
- 使用 `npm test` 来执行单元测试。
- 使用 `npm run autod` 来自动检测依赖更新，详细参见 [autod](https://www.npmjs.com/package/autod) 。
